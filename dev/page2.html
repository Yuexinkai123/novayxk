Can you help me optimize this React component? It's re-rendering too frequently when the parent state changes, even though its own props aren't changing.

smart_toy

Novayxk AI

To prevent unnecessary re-renders in React, you should use React.memo for functional components. If you're passing callbacks or complex objects, ensure they are stabilized using useCallback or useMemo in the parent.

OptimizedComponent.tsx 
import React, { memo } from 'react';

// Wrap your component in memo()
const ExpensiveComponent = memo(({ data, onAction }) => {
  console.log("Rendering...");
  return (
    <div>
      <h2>{data.title}</h2>
      <button onClick={onAction}>Click Me</button>
    </div>
  );
});

export default ExpensiveComponent;

Note: Memoization has a small overhead. Only apply it to components that actually suffer from expensive re-renders or frequent parent updates.
